# 求職者ユーザー登録フロー

## 概要
本ドキュメントでは、求職者が本プラットフォームに登録する際の手順と業務ルールを説明します。

## 登録ステップ

### Step 1: メールアドレス仮登録
- 求職者がトップページから「新規登録」を選択
- メールアドレスを入力して「確認メール送信」をクリック
- システムが仮登録トークン（有効期限: 24時間）を発行
- 確認メールが指定アドレスに送信される

### Step 2: メールアドレス認証
- 受信した確認メール内のリンクをクリック
- トークンの有効性を検証（期限切れの場合はエラー）
- 認証完了後、プロフィール入力画面に遷移

### Step 3: 基本プロフィール入力
必須入力項目：
- 氏名（姓・名）
- 生年月日
- 性別
- 最終学歴
- 現在の就業状況（在学中 / 既卒 / 就業中 / その他）

任意入力項目：
- 電話番号
- 居住都道府県
- 希望職種
- 希望勤務地

### Step 4: 登録完了
- 入力内容を確認画面で確認
- 「登録する」ボタンを押下して本登録完了
- 登録完了通知メールを送信
- ダッシュボードへ自動リダイレクト

## 業務ルール

1. 同一メールアドレスでの重複登録は不可（エラーコード: E_DUPLICATE_EMAIL）
2. 未成年（18歳未満）の登録は保護者同意フローが必要
3. 登録から30日以内にプロフィール補完を促すリマインドメールを送信
4. 仮登録状態が72時間継続した場合、レコードを自動削除

## よくあるエラー

| エラー | 原因 | 対処 |
|--------|------|------|
| E_DUPLICATE_EMAIL | 既登録のメールアドレス | 「ログイン」または「パスワードリセット」へ誘導 |
| E_TOKEN_EXPIRED | 確認メールの期限切れ | 仮登録から再実施 |
| E_INVALID_BIRTHDATE | 不正な生年月日形式 | YYYY-MM-DD形式で入力 |
| E_UNDER_AGE | 18歳未満 | 保護者同意フローへ誘導 |
