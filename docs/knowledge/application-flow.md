# 応募フロー

## 概要

求職者が求人へ応募する際の標準フローを定義します。

## 前提条件

- 求職者がログイン済みであること
- プロフィールの必須項目が入力済みであること
- 対象求人が公開中であること

## 業務ルール

1. 応募時に志望動機の入力を必須とします。
2. 同一求人への重複応募はできません。
3. 応募後は学校側へ通知メールを送信します。
4. 応募済み求人が非公開になった場合も応募履歴は保持します。

## 例外・注意事項

- プロフィール未入力の場合は応募画面へ進めません。
- 募集終了後は応募ボタンを非表示にします。

## FAQ

### Q. 応募の取り消しはできますか。

応募状態が未確認の場合のみ可能です。
