# 学校ユーザー権限マトリクス

## 概要

学校ユーザー向けの主要権限を整理します。

## 前提条件

- 学校組織に所属していること
- ユーザー種別が学校であること

## 業務ルール

1. 管理者はユーザー追加と権限変更が可能です。
2. 編集者は求人の作成と更新が可能です。
3. 閲覧者は応募情報の参照のみ可能です。
4. 個人情報のエクスポートは管理者のみに制限します。

## 例外・注意事項

- 一時権限の付与は運用申請が必要です。
- 権限変更は監査ログを残します。

## FAQ

### Q. 学校管理者が不在の場合はどうしますか。

運用管理チームが暫定管理者を設定します。
