# CSV 取込ルール

## 概要

管理者が CSV で求人またはユーザー情報を一括取込する際のルールです。

## 前提条件

- 指定テンプレートの列順を維持していること
- UTF-8 で保存されていること
- 管理者権限を持つこと

## 業務ルール

1. 必須列が欠けているファイルは取込不可です。
2. 1 行でも致命的エラーがある場合は全件ロールバックします。
3. 取込結果を画面とメールで通知します。
4. 同一キー重複時は更新モードか追加モードを明示します。

## 例外・注意事項

- 改行を含む項目は引用符で囲む必要があります。
- ファイルサイズ上限を超える場合は分割アップロードします。

## FAQ

### Q. 文字化けした場合の最初の確認ポイントは何ですか。

エンコードが UTF-8 かどうかを確認してください。
