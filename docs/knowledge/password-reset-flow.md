# パスワードリセットフロー

## 概要
求職者・学校ユーザー共通のパスワードリセット手順を定義します。

## フロー

### Step 1: リセット申請
- ログイン画面の「パスワードを忘れた方はこちら」をクリック
- 登録済みメールアドレスを入力して送信

### Step 2: メール送信
- リセット用URLをメール送信（有効期限: 60分）
- 存在しないメールアドレスを入力した場合も「送信しました」と表示（アカウント存在有無の漏洩防止）

### Step 3: 新パスワード設定
- メール内URLにアクセス
- 新しいパスワードを2回入力
- 以下のパスワードポリシーを満たす必要あり:
  - 8文字以上
  - 英大文字・小文字・数字をそれぞれ1文字以上含む
  - 直近5回使用したパスワードは再使用不可

### Step 4: 完了
- パスワード変更完了の通知メールを送信
- 全デバイスのセッションを強制ログアウト
- ログイン画面へリダイレクト

## セキュリティ考慮

- リセットURLは使用後に無効化（ワンタイム）
- 同一IPから5回連続でリセット申請を失敗した場合: 30分間リクエストをブロック
- リセット操作はすべてセキュリティログに記録

## よくある問題

| 問題 | 原因と対処 |
|------|----------|
| メールが届かない | 迷惑メールフォルダを確認。未着の場合は10分後に再送可 |
| URLが無効と表示 | 有効期限（60分）切れ。再申請が必要 |
| 新しいパスワードが設定できない | ポリシー未満 or 直近5回と重複 |

## 管理者によるパスワードリセット

管理者は管理画面から対象ユーザーのパスワードを強制リセット可能:
1. ユーザー検索
2. 「パスワードリセット送信」ボタンを押下
3. 対象ユーザーにリセットメールが届く
4. 管理者がパスワードを直接設定することはできない（セキュリティポリシー）
