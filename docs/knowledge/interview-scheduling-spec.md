# 面接日程調整機能仕様

## 概要
学校が求職者に面接候補日を提示し、求職者が希望日を選択する面接日程調整フローを定義します。

## フロー概要

```
学校：書類選考通過 → 面接候補日を提示
          ↓
求職者：候補日から1つ選択 OR 辞退
          ↓
学校：面接確定 → 確定通知
          ↓
当日：面接実施
          ↓
学校：選考結果を登録（内定 / 不合格 / 次選考へ）
```

## 面接候補日の提示

- 学校が最大5つの候補日時を入力
- 各候補に面接形式を設定（対面 / オンライン / 電話）
- 対面の場合: 住所・部屋番号を入力必須
- オンラインの場合: URLまたはミーティングIDを入力必須
- 候補日の有効期限: 提示から5日間

## 求職者の選択

- 候補日一覧から1つを選択（複数選択不可）
- 「辞退する」ボタンから選考辞退も可能
- 選択期限を過ぎた場合（5日経過）: 学校に通知、候補日再提示を促す

## 面接確定後

- 学校・求職者双方にカレンダー添付メールを送信（.ics形式）
- 面接前日に自動リマインドメールを送信
- アプリ内の「面接予定」一覧に表示

## 面接のキャンセル・日程変更

### 学校起因のキャンセル
1. 学校がキャンセル申請（理由入力必須）
2. 求職者に通知
3. 新しい候補日提示フローへ（または選考終了へ）

### 求職者起因のキャンセル
1. 求職者がキャンセル申請
2. 学校に通知
3. 学校が再調整するかどうか選択

### ルール
- 面接24時間前以内のキャンセルは双方に警告表示
- キャンセル回数が3回以上の求職者は管理者に通知

## 選考結果の登録

面接実施後、学校が選考結果を登録:

| 結果コード | 説明 | 求職者への通知 |
|-----------|------|-------------|
| PASS | 内定 | 内定通知メール送信 |
| FAIL | 不合格 | 不合格通知メール送信 |
| NEXT | 次選考へ | 次回面接調整フローへ |
| HOLD | 保留 | 通知なし（1週間後に自動通知） |

## 業務ルール

1. 選考結果は面接日から14日以内に登録必須（14日超過で管理者アラート）
2. 内定後は求職者が承諾 / 辞退を選択できる（期限5日間）
3. 内定承諾後は他の応募中求人に自動で「辞退」処理は行わない（求職者が手動対応）
