# 求職者プロフィール仕様

## 概要
求職者プロフィールの項目定義、公開範囲ルール、および編集制限を定義します。

## プロフィール項目一覧

### 基本情報（必須）

| 項目 | 型 | 制約 | 公開範囲 |
|-----|-----|------|---------|
| 氏名（姓） | text | 最大50文字 | 応募先学校のみ |
| 氏名（名） | text | 最大50文字 | 応募先学校のみ |
| 氏名カナ | text | 最大100文字（全角カナ） | 応募先学校のみ |
| 生年月日 | date | 18歳以上 | 非公開（年齢のみ表示） |
| 性別 | select | 男性 / 女性 / その他 / 回答しない | 設定による |
| 最終学歴 | select | 中卒〜大学院 | 全学校 |
| 就業状況 | select | 在学中 / 既卒 / 就業中 / 求職中 | 全学校 |

### 希望条件（任意）

| 項目 | 型 | 制約 | 公開範囲 |
|-----|-----|------|---------|
| 希望職種 | multiselect | 最大5件 | 全学校 |
| 希望勤務地 | multiselect | 都道府県、複数選択可 | 全学校 |
| 希望年収 | select | 範囲選択 | 応募先学校のみ |
| 就業開始可能日 | date | 今日以降 | 応募先学校のみ |
| リモートワーク希望 | select | 希望する / どちらでも / 希望しない | 全学校 |

### 経歴・スキル（任意）

| 項目 | 型 | 制約 |
|-----|-----|------|
| 職務経歴 | text（複数） | 1件あたり1000文字以内、最大10件 |
| 保有スキル | tag | 最大20個 |
| 資格・免許 | text（複数） | 最大20件 |
| 自己PR | text | 最大2000文字 |
| 履歴書 | file | PDF / Word、10MB以内 |

## 公開範囲設定

求職者は以下の公開範囲を個別に設定可能:

| 設定 | 説明 |
|-----|------|
| 全公開 | すべての学校が閲覧可能 |
| 応募先のみ | 自分が応募した学校のみ閲覧可能 |
| スカウト可・応募先のみ | スカウト受信 + 応募先のみ閲覧可能 |
| 非公開 | いかなる学校も閲覧不可 |

**デフォルト設定**: 応募先のみ

## プロフィール完成度スコア

プロフィールの入力完成度を0〜100でスコア化し、ダッシュボードに表示:

| スコア範囲 | ステータス | 検索への影響 |
|-----------|---------|------------|
| 0〜39 | 未完成 | 検索結果に表示されない |
| 40〜69 | 基本完成 | 通常表示 |
| 70〜89 | 充実 | 検索スコアにボーナス |
| 90〜100 | 完全完成 | スカウト受信優先表示 |

## 編集制限

- 一度応募した後の**氏名・生年月日**の変更は管理者承認が必要
- **性別**の変更は月1回まで
- **プロフィール完成度が40未満**の場合、求人への応募が不可
