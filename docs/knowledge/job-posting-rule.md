# 求人公開ルール

## 概要

学校ユーザーが求人を公開する際の前提条件と入力ルールを定義します。

## 前提条件

- 学校アカウントが審査済みであること
- 有効な担当者が 1 名以上登録されていること
- 学校ユーザー権限を保有していること

## 業務ルール

1. 求人タイトルは必須です。
2. 応募締切日は当日以降でなければなりません。
3. 公開状態にする場合は仕事内容の説明を必須とします。
4. 応募済み求人は削除できず、公開停止のみ可能です。

## 例外・注意事項

- 審査前アカウントは下書き保存のみ可能です。
- 締切日不正の場合はエラーとして差し戻します。

## FAQ

### Q. 公開後に勤務地を修正できますか。

可能ですが、応募者が存在する場合は変更履歴を残します。
